# PWA Icons

Ce dossier contient les icônes pour la Progressive Web App KitiSmart.

## Tailles requises:
- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Génération des icônes:
Vous pouvez générer toutes les tailles à partir d'une icône 512x512 en utilisant:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## Format:
- PNG avec transparence
- Arrière-plan: #0d9488 (couleur primaire KitiSmart)
